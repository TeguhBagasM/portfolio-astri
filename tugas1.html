Algoritma Pencarian: BFS & DFS

Penelusuran graf menggunakan antrian dan tumpukan

BFS

Menelusuri simpul per level dengan Queue (FIFO). Menjamin jalur terpendek (jumlah edge minimum) pada graf tak berbobot.

DFS

Menelusuri sedalam mungkin dengan Stack (LIFO). Hemat memori dan cocok untuk eksplorasi seluruh kemungkinan jalur.

Implementasi Python 

from collections import deque

def bfs(graph, start, goal):
    queue = deque([(start, [start])])
    visited = {start}
    while queue:
        node, path = queue.popleft()
        if node == goal:
            return path
        for nb in graph[node]:
            if nb not in visited:
                visited.add(nb)
                queue.append((nb, path + [nb]))
    return None
def dfs(graph, node, goal, visited=None, path=None):
    if visited is None: visited, path = set(), [node]
    visited.add(node)
    if node == goal: return path
    for nb in graph[node]:
        if nb not in visited:
            r = dfs(graph, nb, goal, visited, path+[nb])
            if r: return r
    return None

Perbandingan BFS vs DFS

Aspek BFS DFS
Struktur Data Queue (FIFO) Stack (LIFO)
Optimalitas ✅ Optimal ❌ Tidak optimal
Kompleksitas O(V + E) O(V + E)
Cocok untuk Jalur terpendek Eksplorasi penuh

Kesimpulan

BFS ideal untuk jalur terpendek menggunakan Queue, sedangkan DFS lebih hemat memori dengan Stack. Pemilihan algoritma bergantung pada kebutuhan: optimalitas (BFS) atau efisiensi memori (DFS).
