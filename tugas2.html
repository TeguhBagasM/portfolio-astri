Minimax & Alpha-Beta Pruning

Algoritma keputusan optimal untuk permainan dua pemain

Minimax

Maximizer (AI) memaksimalkan skor, Minimizer (lawan) meminimalkannya. Menelusuri seluruh pohon kemungkinan untuk keputusan optimal.

Alpha-Beta Pruning

Memangkas cabang tidak relevan dengan batas alpha dan beta. Kompleksitas turun ke O(b^(d/2)) tanpa mengubah keputusan AI.

Info: Alpha-Beta memangkas cabang jika kondisi α ≥ β terpenuhi. Semakin awal pemangkasan terjadi, semakin besar penghematan komputasi.

Kode Alpha-Beta Pruning (Python) 

def minimax(board, depth, alpha, beta, is_max):
    score = evaluate(board)
    if score != 0 or not moves_left(board):
        return score
    if is_max:
        best = -1000
        for i, j in empty_cells(board):
            board[i][j] = 'O'
            best = max(best, minimax(board, depth+1, alpha, beta, False))
            board[i][j] = '_'
            alpha = max(alpha, best)
            if beta <= alpha: break  # ✂ pruning
        return best
    else:
        best = 1000
        for i, j in empty_cells(board):
            board[i][j] = 'X'
            best = min(best, minimax(board, depth+1, alpha, beta, True))
            board[i][j] = '_'
            beta = min(beta, best)
            if beta <= alpha: break  # ✂ pruning
        return best

Kesimpulan

Minimax menghasilkan keputusan optimal pada permainan dua pemain. Alpha-Beta Pruning meningkatkan efisiensi secara signifikan dari O(b^d) menjadi O(b^(d/2)) tanpa mengubah hasil keputusan AI.
